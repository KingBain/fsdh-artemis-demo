4
This is the architecture summary slide. Explain that OEM means Orbit Ephemeris Message, a text-based space data format that contains timestamps plus position and velocity state vectors. Say that the API portion of the solution fetched the live data source, saved it into the landing area, then the notebooks processed it through Bronze, Silver, and Gold. Bronze is raw ingestion: file-level and line-level capture, so we preserve the original source. Silver is parsed and structured data: metadata, timestamps, position, velocity. Gold is business-ready or presentation-ready data: speed over time, mission duration, latest known state, summary metrics, and views for the site.
